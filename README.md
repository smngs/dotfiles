Dotfiles
---

## Usage (Client)

``` shell
$ git clone https://git.mine-313.com/syota/dotfiles.git
$ cd dotfiles
$ make install
```

## Usage (Server/Docker)

``` shell
$ git clone https://git.mine-313.com/syota/dotfiles.git
$ cd dotfiles
$ make install-minimal
```

## Require
### Minimal
- git
- zsh
- emacs

### Maximal
- i3
- polybar
